# Getting started

## Installation
### npm
<pre class="code-line">npm install --save faanshu-ui</pre>
### yarn
<pre class="code-line">yarn add faanshu-ui</pre>

## Usage
Wrap your app with the faanshu-ui provider, this ensures the ui components render with consistent styles.
```
import { FsProvider } from 'faanshu-ui';

const App = () => {
  return (
      <FsProvider>
        <MyApp />
      </FsProvider>
  )
}
```
Use faanshu-ui components anywhere in your app.
```
import { Header } from 'faanshu-ui';

const App = () => {
  return (
    <Header>Hello from faanshu-ui!</Header>
  )
}
```
